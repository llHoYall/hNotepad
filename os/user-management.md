# User Management

### 사용자 추가

- `useradd` : 새로운 사용자를 만들거나 기존 사용자의 정보를 업데이트 한다.
- `adduser` : 사용자를 시스템에 추가한다.

상세하게 옵션 설정을 해서 추가를 하려면 `useradd` 명령어를 사용하고, 간단하게 기본 설정으로 추가하려면 `adduser` 명령어를 사용한다.

> $ adduser hoya
