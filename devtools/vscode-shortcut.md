# Visual Studio Code - Shortcut

### 커서

- `<CTRL> + <ALT> + <SHIFT> + <UP/DOWN>` : 위/아래에 커서 추가. (인텔 내장 그래픽과의 충돌 때문에 \<SHIFT>추가해야 정상 동작함.)
- `<ALT> + <LMB>` : 해당 위치에 커서 추가.

### 검색

- `<CTRL-f>` : 현재 파일 내에서 검색.
- `<CTRL> + <SHIFT-f>` : 현재 프로젝트 내에서 검색.
- `<ALT> + <UP>` : 이전 검색 히스토리.
- `<ALT> + <DOWN>` : 다음 검색 히스토리.


### 선택

- `<CTRL-d>` : 커서위치부터 같은 단어를 선택.
