# DataBase - Basic

## Query Statements

```sql
CREATE TABLE [table] (data)
INSERT INTO [table] [(columns...)] VALUES ([data])
UPDATE [table] SET [column] = [data] WHERE [condition]
DELETE FROM [table] WHERE [condition]
SELECT [column or *] FROM [table] WHERE [condition]
```
