# [Plugin] Netrw

vim에서 파일대신 디렉토리를 열면 자동으로 이 플러그인으로 디렉토리 목록이 보인다.

예를 들면 다음과 같다.

- `:e .`, `:edit .`
- `:sp .`, `:split .`
- `:vs .`, `:vsplit .`
- `:E`, `:Explore`
- `:Se`, `:Sexplore`
- `:Vex`, `:Vexplore`



자주 사용되는 단축키는 다음과 같다.

- `<CR>` : Browsing.
- `o` : 수평 분할 창으로 연다.
- `t` : 새 탭으로 연다.
- `v` : 수직 분할 창으로 연다.
- `i` : 보기 방식을 바꾼다.
- `D` : 파일 혹은 디렉토리를 삭제한다.
- `-` : 상위 디렉토리로 이동한다.
- `d` : 새 디렉토리를 생성한다.
- `%` : 새 파일을 생성한다.
- `p` : 수평 분할 창에 미리보기 한다.
- `P` : 이전 창에 연다. 이전 창이 없을 경우 수평 분할 창에 연다.
- `r` : 역순으로 정렬한다.
- `R` : 파일 혹은 디렉토리의 이름을 변경한다.
- `s` : 정렬 방식을 변경한다.
