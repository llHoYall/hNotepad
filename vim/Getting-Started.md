# Getting Started

### Installation

- on MAC

  > $ brew install vim

- on Linux

  > $ apt-get install vim

&nbsp;

### Run

- with default setting file (.vimrc)

  > $ vim

- with factory settings

  >  $ vim -u NONE -N

- with custom setting file

  >  $ vim -u {*custom\_setting\_file*}.vim
