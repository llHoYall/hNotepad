# Powershell Cmdlet

- **Get-Help**
- **Copy-Item** {*from*} {*to*} : cpi, cp, copy
  -Recurse
- **New-Item** {path} : ni
  -Type {directory|file}
  -Force
  -Value "*data*"
- **Remove-Item** {*path*} : ri, rd, erase, rm, rmdir, del
  -Recurse
  -Force
  -Include {*file*}
  -Exclude {*file*}
- **Rename-Item** {*current*} {*new*} : rni, ren
- **Set-Alias** {*name*} {*path*}
- **Test-Path** {*path*}
  return True|False
