# fasd

사용빈도가 높은 파일 또는 디렉토리 검색을 편하게 해준다.

#### Installation

-  MAC

> $ brew install fasd

#### Setting

zsh에 설정을 추가한다.

```shell
# .zshrc

plugins=(
	...
	fasd
	...
)
```

#### Usage

- `z` : 디렉토리 이동
- `s` : 파일 or 디렉토리 검색
